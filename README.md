# Olivin
[KDT] 프로젝트 중심(MES, SCM, 구독형 ERP) Java 풀스택 웹 개발자 양성과정 (3회차) - 기업요구 MES 프로젝트

## 프로젝트 개요
Olivin은 올리브 영의 유통망에서 모티브를 따온 SCM 프로젝트입니다.
공급 업체, 본사, 지점 간의 물류 흐름과 재고 현황을 파악할 수 있도록 지원합니다.
